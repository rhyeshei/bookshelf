# 本棚アプリ（Book Review App）

## 概要
本棚アプリは、**書籍の登録・閲覧・レビュー投稿**が可能なDjango製Webアプリケーションです。  
『**Djangoのツボとコツがゼッタイにわかる本［第3版］**』の学習を通じて、Djangoの基本構造とWebアプリ開発フローの理解を目的として開発しました。

---

## 主な機能

### ユーザー認証
- 会員登録
- ログイン / ログアウト

### 書籍管理
- 書籍一覧表示・詳細表示  
- 新規書籍の登録  
- 登録済み書籍の編集・削除  
  - ※ 投稿ユーザー本人のみ可能

### レビュー機能
- 書籍に対する **5段階評価**
- コメント付きレビュー投稿

### 表示機能
- 評価順ランキング表示（トップページ）
- ページネーション対応

---

## 技術スタック
- **Framework**: Django 4.2.26  
- **Language**: Python 3.9.6
- **Database**: SQLite  
- **Frontend**: Bootstrap 5.3.0  

---

## 環境構築・実行方法

### 1. リポジトリをクローン
```bash
git clone <repository-url>
cd bookproject
```

### 2. マイグレーションの実行
※ 初回起動時はマイグレーションの実行が必須です。
```bash
python manage.py migrate
```

### 3. 開発サーバーの起動
```bash
python manage.py runserver
```
